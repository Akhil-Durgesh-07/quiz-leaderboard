# quiz-leaderboard

This project polls a quiz api for 10 times and it generates the leaderboard

PROCESS:
1. calling api 10times with 5seconds deplay
2. removes duplicate data using (roundid + participant)
3. use hashmap to store total scores
4. sorted the leaderboard in descending order
5. submitted final result via POST API method

OUTPUT:
Diana - 470
Ethan - 455
Fiona - 440

Total score - 365
